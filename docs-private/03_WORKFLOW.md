# SNSアプリ 開発ワークフロー (Workflow & Checklists)

## 開発ルール (Rules for Execution)

### 動作確認ルール (Verification Standards)

ロードマップの各タスクを「完了」とする前に、**必ず** 以下の検証を行い、正常動作を確証すること。

- **バックエンド (Supabase / DB):**
    - **データ整合性:** DBの中身を直接確認し、データが正しく保存・更新されているか。
    - **型定義とRLS:** `database.types.ts` の型と実際のデータ構造が一致しているか。RLS (Row Level Security) により不正なアクセスが弾かれているか。
    - **API機能:** Supabase Client 経由でのリクエストに対して、正しいデータまたはエラー（PostgRESTエラー等）を返しているか。
- **フロントエンド (Next.js/UI):**
    - **型安全:** ZodバリデーションおよびReact Queryからのデータ型が正しく認識されているか。
    - **状態管理:** Zustandストアのグローバルな状態が予測通りに更新されているか。
    - **楽観的UI:** いいね処理などのAPIレスポンスを待たないUI更新が正常に機能しているか。
- **環境整合性の検証 (Environment Parity):**
    - **Local Production Build:** リリース前には必ず `npm run build && npm run start` を実行し、本番モード (`NODE_ENV=production`) で動作確認を行うこと。特に画像最適化やビルド時の型エラーをこの段階で完全に排除する。
- **ネットワーク通信:**
    - Networkタブでブラウザへ送信されるデータの最適化（過剰なリレーション先の読み込み防止など）がされているか。
    - 本番モードにおいて、Next.js による画像最適化がエラーなく機能し、適切なサイズで配信されているか。

### DB 変更の標準サイクル (DB Migration Workflow)

スキーマ（テーブル、カラム、RLS）を変更する際は、必ず以下の手順を遵守すること。

1. **SQL 作成:** `supabase/migrations/` 内に新規マイグレーションファイル（`.sql`）を作成する。
2. **反映:** `npx supabase db push` コマンドでクラウド上のデータベースに反映させる。
3. **型同期:** `npm run codegen` を実行し、変更後の構造に基づいた `database.types.ts` を再生成する。
4. **実装:** 更新された型定義を用いてフロントエンドの実装・修正を行う。

### プロセス記録ルール (Mandatory)

`git push` を実行する直前に、**必ず** 以下の手順を実行すること。

#### ステップ 1: 開発ログの記録

- **対象ファイル:** `docs-private/07_DEV_LOG.md`
- **アクション:** 実装内容、技術的判断、エラーと解決策を追記する。（`write_file`ではなく`replace`による追記を徹底すること）

#### ステップ 2: 復旧マニュアルの整合性確認

- **対象ファイル:** `docs-private/01_AGENT_RECOVERY_MANUAL.md`
- **アクション:** `07_DEV_LOG.md` の更新内容に基づき、ステータスを最新化する。

#### ステップ 3: ワークフローの更新

- **対象ファイル:** `docs-private/03_WORKFLOW.md`
- **アクション:** 完了したタスクのチェックボックスをオンにし、「現在進行中のタスク」を更新する。

#### ステップ 4: Gitコミット

- **言語:** 日本語で記述すること (`feat:`, `fix:`, `docs:`, `chore:`, `refactor:` プレフィックスを使用)。
- **AI 共同著作者の明記 (Findy 対策):** AI エージェント（Gemini CLI 等）が作成したコミットには、必ず末尾に以下の署名（空行を挟む）を含めること。
  ```text
  Co-authored-by: Gemini CLI <gemini-cli@google.com>
  ```

開発スピードを維持しつつ、品質を担保するためのフロー。
